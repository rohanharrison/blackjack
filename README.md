# Blackjack Game
To run this app:
Make sure Nodejs and MongoDB is installed.
Create a directory /data/db for MongoDB (C:/ on my computer).
```console
$ git clone https://github.com/rohanharrison/Blackjack
$ cd Blackjack
$ npm install
$ mongod &  # start mongodb
$ npm start      # host locally
```
